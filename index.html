All representations were produced from the MASTER file with the following FFMPEG parameters:
— Framerate not defined (inherited from the uploaded source).
— Codec: libx264.
— Forced keyframes = 3 seconds (considering the further DASH segment duration).
— Bitrate (VBR) defined through CRF.
— -sc_threshold 0 was not defined. (to keep "organic" keyframes based on scenes as-is. The segmenter's cutting with the defined time interval works perfectly with the forced keyframe interval).
BTW: I tried to pass all MP4Box operations with this "classical" flag, but it didn't help.
